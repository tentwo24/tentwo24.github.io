2.颜色的alpha通道

颜色的alpha通道标识了色彩的透明度，它是一个0~1之间的取值，0标识完全透明，1标识完全不透明

在css中使用rgba可以为颜色添加alpha通道

rgba还可以有多种书写方式，例如rgba(0, 0, 0, 0.5)还可以写为：

  • rgb(0 0 0 / 50%)
  • #00000080

3.尺寸的百分比

绝大部分可以书写尺寸的地方，都可以书写百分比

百分比是一个相对单位，其相对于元素的参考系，比如：

  • 普通元素的参考系为父元素的内容区域（即只有content部分）
  • 绝对（固定）定位元素的参考系为父元素中第一个定位元素的padding区域（既包括content内容也包括padding）

下面罗列常见的百分比情况

css属性 百分比相对于 备注
width 参考系的宽度
height 参考系的高度 参考系高度受本身宽度影响时，设置无效
padding 参考系的宽度
border 参考系的宽度
margin 参考系的宽度

4.最大最小宽高

  • 最大宽度：max-width，最大高度：max-height
  • 最小宽度：min-width，最小高度：min-height

当一个元素的尺寸会自动变化时，设置最大最小宽高，可以让它不至于变得过小或过大。

在实际开发中，我们通常为PC端的页面设置一个最小宽度，通常此宽度为设计稿的宽度

又或者，我们会为页面中的所有图片设置一个最大宽度，让其不至于超过容器

三、 表单进阶-CSS

1.CSS中的 outline 和 border 属性

​ outline 属性用于在元素周围创建一个轮廓，类似于边框。然而，outline 和 border 之间最主要的区别是，outline 并不占据布局空间，不改变元素的大小和位置。outline 主要用于焦点、高亮和可见性控制。

outline 可以有以下属性：

  • outline-color：指定轮廓的颜色；
  • outline-style：指定轮廓的样式，比如实线、虚线等；
  • outline-width：指定轮廓的宽度。

​ border 属性用于定义元素的边框，同时边框将占据布局空间。border 通常用于元素的美化、布局和结构化。

border 可以有以下属性：

  • border-color：指定边框的颜色；
  • border-style：指定边框的样式，比如实线、虚线等；
  • border-width：指定边框的宽度；
  • border-radius：指定边框的圆角半径。

​ 尽管 outline 和 border 都可以创建元素的边框效果，但它们之间有几个关键的区别：

  1. 占据布局空间不同：outline 不占据布局空间，而 border 占据布局空间。这意味着当为元素设置 outline 时，它的大小和位置不会改变，而设置 border 时，元素的大小和位置会相应改变。
  2. 可见性不同：outline 可以独立于元素内容显示或隐藏，它主要用于用户交互和焦点控制。而 border 组成了元素的实际可见边界。
  3. 更多样式控制：border 提供了更多的样式控制选项，如 border-radius 属性用于创建圆角边框。
  4. **常见使用方法 **：通常点击input聚焦的时候会出现一个outline外边框

2.细节处理

  • 处理textarea标签不允许拖动改变文本域大小，resize属性

    • none：用户不能操纵机制调节元素的尺寸；
    • both：用户可以调节元素的宽度和高度；
    • horizontal：用户可以调节元素的宽度；
    • vertical：让用户可以调节元素的高度；
    • inherit：默认继承。

  • 伪类（处理各种状态的元素）

    • :focus 表示获得焦点的元素（如表单输入）。当用户点击或轻触一个元素或使用键盘的 Tab 键选择它时，它会被触发。
    • :disabled 表示任何被禁用的元素。如果一个元素不能被激活（如选择、点击或接受文本输入）或获取焦点，则该元素处于被禁用状态。
    • :checked 表示任何处于选中状态的radio(<input type="radio">), checkbox (<input type="checkbox">) 或 (“select”) 元素中的option HTML 元素 (“option”)。

  • 伪元素

    • ::placeholder 表示input或textarea元素中的占位文本，可以通过这个修改占位文本的样式

  • 鼠标样式cursor

    • help 问号
    • wait loading
    • crosshair 十字选择器
    • not-allowed 禁用
    • zoom-in 放大器
    • grab 小手

  • 选择器

    • ~ 组合器选择兄弟元素，也就是说，后一个节点在前一个节点后面的任意位置，并且共享同一个父节点。

      • 例子：p ~ span 匹配同一父元素下，p元素后的所有 span元素。

    • + 组合器选择相邻元素，即后一个元素紧跟在前一个之后，并且共享同一个父节点。

      • 例子：h2 + p 会匹配紧邻在h2元素后的第一个p元素。

    • > 组合器选择前一个元素的直接子代的节点。

      • 例子：ul > li 匹配直接嵌套在ul元素内的所有li 元素。

  • 了解

    • vertical-align 用来指定行内元素（inline）或表格单元格（table-cell）元素的垂直对齐方式。只对行内元素、行内块元素和表格单元格元素生效，不能用它垂直对齐块级元素。主要用于两个行内元素垂直对齐
    • 所有绝对定位和浮动的元素会自动变成 display:block

1.什么是精灵图

​ 子画面是二维图像，由在定义的X和Y坐标下将小图像组合成一个大图像组成。要显示组合图像中的单个图像，可以通过使用CSS background-position 属性，定义要显示的图像的确切位置。(详情看这里：CSS Sprites(精灵图) - 菜鸟教程 (cainiaojc.com))

2.精灵图的优点及为什么要使用精灵图

  • 采用了精灵图这一技术可以缓解加载时间过长从而影响用户体验的这个问题。
  • 在一定程度上减少了页面的加载速度，也一定程度上缓解了服务器的压力。
  • 使用精灵图可以有效减少服务器接收和发送请求的次数，从而达到提高页面的加载性能。

六、拓展知识

1. 伪类选择器

css伪类选择器大全：

https://developer.mozilla.org/zh-CN/docs/Web/CSS/Pseudo-classes

伪类 含义
:link 选中未访问过的超链接
:visited 选中已访问过的超链接
:hover 选中鼠标移入的元素
:active 选中鼠标按下的元素
:focus 选中聚焦的表单元素
:disabled 选中被禁用的表单元素
:checked 选中被选中的表单元素
:first-child 选中第一个子元素
:last-child 选中最后一个子元素
:nth-child(an+b) 选中第「an+b」个子元素
a和b是常量，n的值会从0开始依次递增
:first-of-type 选中第一个指定类型的子元素
:last-of-type 选中最后一个指定类型的子元素
:nth-of-type(an+b) 选中第「an+b」个指定类型的子元素
a和b是常量，n的值会从0开始依次递增

注意:使用:link;:visited;:hover;:active这四个伪类的时候需要遵循爱恨法则。根据css属性值的计算过程中层叠冲突中的比较源次序，不按照顺序来写的话，会影响用户体验，如果先写:active,再写:visited，按下的时候显示的就是 选中已访问过的超链接设置的样式。
